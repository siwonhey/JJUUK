# 쭉(JJUUK) 사용 가이드

안녕하세요, 쭉이에요.

오래 앉아 일하다 보면 어느새 어깨가 말리고 목이 앞으로 나오죠. 매번 의식하기는 어려운 일이에요. 그래서 쭉이 곁에서 살짝씩 알려드리려고 해요. 자세가 흐트러진 순간에만 거북이나 기린 친구가 슬쩍 등장해서 신호를 보내요. 그게 전부예요. 무겁지 않게, 부드럽게.

이 안내서 한 장이면 시작하실 수 있어요.

---

## 1. 설치하기

받으신 **`JJUUK Setup 0.1.0.exe`** 파일을 더블클릭해 주세요. 별도 설정 없이 사용자 폴더에 자동 설치돼요. 관리자 권한도 필요 없어요.

**혹시 "Windows 가 PC 를 보호했습니다" 라는 파란 화면이 떴다면**, 당황하지 않으셔도 돼요. 쭉은 아직 정식 코드 서명 절차를 거치지 않은 사내 배포본이라 Windows 가 처음 보는 프로그램으로 인식한 거예요. 화면의 작은 **"추가 정보"** 글자를 누른 다음, 아래에 나타나는 **"실행"** 버튼을 눌러 주세요. 한 번만 거치면 다음부터는 그냥 열려요.

---

## 2. 처음 만나기

설치가 끝나면 쭉이 인사 화면과 함께 등장해요.

1. 화면을 한 번 훑어보시고 **시작하기** 를 눌러 주세요.
2. Windows 가 "카메라를 사용해도 될까요?" 라고 물어보면 **허용** 을 눌러 주세요.
3. 카메라가 켜지면, 화면 중앙의 네모 안에 얼굴이 들어오도록 평소처럼 모니터 보는 자세로 편하게 앉아 주세요.
4. **측정 시작** 버튼을 누르면 3초 동안 자세를 기억해요. 이때는 그대로 가만히 계시면 돼요.
5. 마지막에 **이대로 좋아요** 를 누르면 끝이에요. 쭉은 슬그머니 트레이로 자리를 옮겨요.

이 3초가 앞으로 여러분의 자세를 비교할 기준이 돼요. 정자세로 앉아 측정하실수록 정확해져요.

---

## 3. 일상 속에서 쓰기

쭉은 화면 오른쪽 아래(맥은 위쪽 메뉴 바) 시계 옆에 작은 거북이 아이콘으로 머물러 있어요.

- **자세가 흐트러지면** 화면에 거북이(거북목)나 기린(굽은 등) 친구가 슬쩍 나타나요. 자세를 바로잡으면 알아서 사라져요.
- **트레이 아이콘을 클릭하면** 켜기·끄기 토글, 자세 다시 측정, 설정 메뉴가 작은 카드로 떠요.
- **우클릭(맥은 두 손가락 클릭)** 하면 종료를 포함한 전체 메뉴가 보여요.
- **설정 창** 에서는 캐릭터 디자인을 바꾸고, 알림 크기(보통·작게)와 감지 민감도, 다크 모드, 자동 실행을 조정하실 수 있어요. 오늘 하루 자세가 어땠는지 한눈에 보여주는 작은 리포트도 함께 있어요.

기준이 잘 안 맞는다 싶으시면 언제든 **자세 다시 측정** 으로 새로 잡으시면 돼요. 자리를 옮겼거나, 모니터 위치가 바뀌었거나, 측정한 지 2주가 넘었다면 한 번 다시 잡아 주시는 걸 권해드려요.

## 4. 자주 묻는 질문

**카메라 LED 가 켜져 있어요. 영상은 어디로 가나요?**
어디에도 가지 않아요. 영상은 여러분 컴퓨터 안에서만 분석되고, 분석이 끝난 프레임은 그 즉시 메모리에서 사라져요. 어떤 서버에도 전송되지 않고, 한 장도 저장되지 않아요.

**잠시 쉬고 싶어요.**
트레이 아이콘을 클릭해서 **켜기 토글을 끄시면** 돼요. 그러면 카메라 트랙이 완전히 해제되어 LED 도 함께 꺼져요. 다시 켜는 것도 같은 방식이에요.

**알림이 너무 자주 떠요 / 거의 안 떠요.**
설정 창에서 **감지 민감도** 를 조절해 주세요. **여유롭게 / 보통 / 꼼꼼하게** 중에서 고르실 수 있어요. 잘 모르시겠다면 "보통" 을 권해드려요.

**부팅하면 자동으로 실행되게 하고 싶어요.**
설정 창에서 **컴퓨터 켤 때 자동 실행** 토글을 켜 주세요.

**여러 모니터를 쓰는데, 보조 모니터에서도 알림이 보이나요?**
네, 모든 모니터에 동시에 표시돼요. 모니터를 새로 연결하거나 분리해도 자동으로 따라가요.

**다른 사람이 잠깐 자리에 앉으면 어떻게 되나요?**
쭉은 화면에서 가장 크게 잡힌 얼굴, 즉 카메라에 가장 가까운 분(=본인) 을 기준으로 판단해요. 옆에 누가 잠깐 지나가도 영향받지 않아요.

## 5. 개인정보에 대한 약속

쭉이 다루는 데이터는 단순해요.

- 카메라 영상은 **여러분 컴퓨터 안에서만** 분석돼요. 인터넷으로 어디에도 전송되지 않아요.
- 분석이 끝난 프레임은 그 즉시 메모리에서 사라져요. **단 한 장도 저장되지 않아요.**
- 컴퓨터에 남는 것은 **자세 기준값(숫자 몇 개)** 과 **일별 자세 통계(상태별 누적 시간)** 뿐이에요.
- 일시정지 시에는 카메라 자체가 꺼져 LED 도 함께 꺼져요.

이 약속은 쭉이 작동하는 모든 순간에 지켜져요.
